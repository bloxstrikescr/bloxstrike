## 🛠️ Troubleshooting Common Issues

- ❌ **Script Does Not Load**
  The executor may be outdated or not properly attached to the Roblox process.
  **Solution:** Update your executor to the latest version, reattach to Roblox, and try loading Bloxstrike Script again.

- ❌ **Hitbox Feature Not Working**
  Bloxstrike Script Hitbox expansion requires the game to be fully loaded before activation.
  **Solution:** Wait until you are fully spawned in a match, then toggle the hitbox feature using `F1` or the GUI.

- ❌ **GUI Does Not Appear**
  The overlay might be blocked by another application or the executor's rendering mode.
  **Solution:** Try running the executor as administrator and ensure no screen overlay apps conflict with Bloxstrike Script.

- ❌ **Hvh Mode Feels Unresponsive**
  High ping or unstable connections can interfere with Bloxstrike Script Hvh resolver calculations.
  **Solution:** Switch to a lower-latency server and verify that `hvh_mode` is set to `true` in your `config.json`.

- ❌ **Config Resets After Restart**
  The `config.json` file may not have proper write permissions on your system.
  **Solution:** Right-click the Bloxstrike Script folder, go to **Properties > Security**, and grant full write access.

- ❌ **FPS Drops When ESP Is Active**
  ESP rendering on older hardware can cause noticeable frame drops.
  **Solution:** Reduce `esp_distance` value in the configuration or disable ESP during heavy combat scenes.
